Что нужно

Docker Desktop — docker.com
Папка с проектом


Первый запуск
bashcd ~/Downloads/3D-Calculator
cp .env.example .env
docker compose up --build
Первый раз качает образы — минут 10. Когда в терминале появится Application startup complete и celery ready — открывай браузер:
http://localhost:3000
В следующий раз просто docker compose up. Остановить — Ctrl+C.

Работа с программой
Зарегистрируйся, создай проект, дай ему название.
На странице проекта три колонки — слева параметры, в центре модель, справа цена.
Загрузка модели — кликни на область загрузки слева, выбери STL/OBJ/3MF файл. Подожди пару секунд, модель появится в центре. В папке samples/ есть два тестовых файла если нет своих.
Параметры — выбери технологию, материал, время печати, количество. Цена справа меняется сама.
AI текст — фиолетовая кнопка внизу справа, генерирует описание детали для клиента.
Всё сохраняется само. Закрыл браузер, открыл снова — всё на месте.

Передача через Google Drive
Сожми папку 3D-Calculator в ZIP, загрузи в Drive, открой доступ по ссылке.
Получателю нужно: скачать, разархивировать, установить Docker Desktop, и выполнить три команды выше. Python, Node.js и базы данных устанавливать не нужно.

Частые проблемы
docker: command not found — Docker Desktop не запущен, открой его.
Модель застряла в processing — docker compose logs worker покажет что случилось.
Сайт не открывается — подожди 30 секунд после запуска и обнови страницу.
